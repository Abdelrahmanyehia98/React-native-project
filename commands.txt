/*
Important Props for <View> in React Native:

⿡ style  
   - Defines styles for the View (background color, padding, margin, etc.).
   - Example:
     style={{ backgroundColor: "blue", padding: 20 }}

⿢ flex (Used Inside style)  
   - Controls how the View expands inside its parent.
   - flex: 1 → Takes all available space.
   - justifyContent → Aligns children vertically ("center", "space-between", etc.).
   - alignItems → Aligns children horizontally.
   - Example:
     style={{ flex: 1, justifyContent: "center", alignItems: "center" }}

⿣ onLayout  
   - Detects when the View is rendered and gets its dimensions.
   - Example:
     onLayout={(event) => { console.log(event.nativeEvent.layout.width) }}

⿤ pointerEvents  
   - Controls whether the View should receive touch events.
   - "none" → Ignores all touches.
   - "box-none" → Parent ignores touch, but children still receive them.
   - "box-only" → Only parent receives touch, not children.
   - "auto" → Default behavior.
   - Example:
     pointerEvents="none"

⿥ accessibilityLabel  
   - Adds a label for screen readers (important for accessibility).
   - Example:
     accessibilityLabel="Profile Picture"

⿦ collapsable  
   - Improves performance by removing Views that don't affect layout.
   - Example:
     collapsable={false}

⿧ hitSlop  
   - Expands the touch area of a View.
   - Example:
     hitSlop={{ top: 20, bottom: 20, left: 20, right: 20 }}
*/

Important Props for <Text> in React Native:

   - Example:
     style={{ color: "blue", fontSize: 18, fontWeight: "bold" }}

   - Handles touch interactions (like making text clickable).

⿥ selectable  
   - Allows the user to select/copy text.
   - Example:
     selectable={true}

⿦ onLongPress  
   - Handles long-press interactions.
   - Example:
     onLongPress={() => console.log("Text long-pressed!")}

⿧ adjustsFontSizeToFit  
   - Automatically shrinks text to fit within its container.
   - Example:
     adjustsFontSizeToFit={true}

⿨ minimumFontScale  
   - Works with adjustsFontSizeToFit to define the smallest font scale allowed.
   - Example:
     minimumFontScale={0.5}

⿩ allowFontScaling  
   - Enables/disables dynamic text scaling based on system settings.
   - Example:
     allowFontScaling={false}

🔹 Example Usage:

  numberOfLines={1}
  ellipsizeMode="tail"
  onPress={() => console.log("Text clicked!")}

  This is an example text.
</Text>

⿡ source  
   - Defines the image source (local or remote).
   - Example (local):
     source={require("./assets/image.png")}

     source={{ uri: "https://example.com/image.jpg" }}

   - Options: "cover" (default), "contain", "stretch", "repeat", "center".

     resizeMode="contain"

⿤ defaultSource (iOS only)  
   - Displays a placeholder image while loading a remote image.
   - Example:
     defaultSource={require("./assets/placeholder.png")}

⿥ onLoad  
   - Called when the image is successfully loaded.
   - Example:
     onLoad={() => console.log("Image loaded!")}

⿦ onError  
   - Called when the image fails to load.
   - Example:
     onError={() => console.log("Failed to load image")}

⿧ fadeDuration (Android only)  
   - Sets the fade-in animation duration (default is 300ms).
   - Example:
     fadeDuration={500}

⿨ blurRadius  
   - Applies a blur effect to the image.
   - Example:
     blurRadius={5}

⿩ progressiveRenderingEnabled (Android only)  
   - Loads a lower-quality version first, then the full image.
   - Example:
     progressiveRenderingEnabled={true}

🔹 Example Usage:
<Image
  source={{ uri: "https://example.com/image.jpg" }}
  style={{ width: 150, height: 150, borderRadius: 10 }}
  resizeMode="cover"
  onLoad={() => console.log("Image loaded!")}
  onError={() => console.log("Failed to load image")}
/>

Important Information About StyleSheet in React Native:

⿡ Importing StyleSheet  
   - To use styles, first import StyleSheet from React Native.
   - Example:
     import { StyleSheet } from "react-native";

⿢ Creating Styles  
   - Use StyleSheet.create to define styles.
   - Example:
     const styles = StyleSheet.create({
       container: {
         flex: 1,
         backgroundColor: "white",
         padding: 20,
       },
       text: {
         fontSize: 18,
         color: "black",
         fontWeight: "bold",
       },
     });

⿣ Applying Styles  
   - Use the style prop to apply styles.
   - Example:
     <View style={styles.container}>
       <Text style={styles.text}>Hello, World!</Text>
     </View>

⿤ Inline Styles vs. StyleSheet  
   - Inline styles:
     <Text style={{ fontSize: 20, color: "blue" }}>Hello</Text>
   - StyleSheet (better performance):
     <Text style={styles.text}>Hello</Text>

⿥ Multiple Styles  
   - You can apply multiple styles using an array.
   - Example:
     <Text style={[styles.text, { color: "red" }]}>Styled Text</Text>

⿦ Common Style Properties  
   - ✅ Layout: flex, justifyContent, alignItems, alignSelf
   - ✅ Spacing: margin, padding, gap
   - ✅ Size: width, height, maxWidth, maxHeight
   - ✅ Borders: borderWidth, borderColor, borderRadius
   - ✅ Colors: backgroundColor, color, opacity
   - ✅ Typography: fontSize, fontWeight, textAlign, lineHeight
   - ✅ Shadow (iOS & Android): shadowColor, shadowOffset, shadowOpacity, shadowRadius (iOS), elevation (Android)

⿧ Dynamic Styles  
   - Styles can be dynamic by using state.
   - Example:
     const [isActive, setIsActive] = useState(false);
     <Text style={[styles.text, isActive && { color: "green" }]}>Dynamic</Text>

🔹 Example Usage:
import { StyleSheet, View, Text } from "react-native";

const App = () => {
  return (
    <View style={styles.container}>
      <Text style={styles.text}>Hello, World!</Text>
    </View>
  );
};

const styles = StyleSheet.create({
  container: {
    flex: 1,
    justifyContent: "center",
    alignItems: "center",
    backgroundColor: "#f5f5f5",
  },
  text: {
    fontSize: 20,
    fontWeight: "bold",
    color: "black",
  },
});

export default App;
